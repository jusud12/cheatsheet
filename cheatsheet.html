HTML Cheat Sheet References

Tags Names Description
<hr> Horizontal row The horizontal rule adds line to further divide segment of the page.
<p> Paragraph The p tag represents the paragraph in the html; it is responsible to create paragraph.
<header> Header The header is an html tag that represents an introductory content especially so for an introductory or navigational aids.
<tbody> Table body The table body encapsulates a set of table row indicating the they comprise of the table body.
<tfoot> Table foot The tfoot element is a set of rows that summarizes the columns of the table
<br> Line break The line break creates new line desired by the users on the page.
<thead> Table head The thead explains the set of row defining the column of the table.
Attribute Name Description
colspan Column span The colspan tells us how many column a table should span. These attributes are essential to building a table.
rowspan Row span The rowspan defines the number of a cell should span. With out the row in the table, it would be difficult to keep track of the columns. It is worth knowing the important of havinfg the row of a table.
cols Column The column is used to hold data in the table.
